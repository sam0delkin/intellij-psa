Invokes Project Specific Autocomplete (PSA) contextual actions for the current file.

Shows a popup with PSA editor actions that have contextual: true in the script response and whose path_regex (if set) matches the current file's directory.

Actions can be configured in the PSA script by returning them in the editor_actions array with the contextual flag set to true: 


{
  "editor_actions": [
    {
      "name": "my_action",
      "title": "My Action",
      "contextual": true,
      "source": "editor",
      "target": "editor"
    }
  ]
}

Actions with a group_name are shown as sub-groups inside the popup. To restrict an action to specific directories, set the path_regex field.

Configure the PSA script path in Settings | Tools | Project Specific Autocomplete .
